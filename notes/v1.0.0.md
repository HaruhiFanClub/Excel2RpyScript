## v1.0.0 正式版

Excel2Rpy v1.0.0 带来完整的视觉小说脚本工作台：表格编辑、资源预览与导入、脚本生成、语音合成、工程同步、检查和对比都集中在一个应用里完成。无论是从 Excel 编排演出，还是把生成结果落实到 Ren'Py 工程，这一版都提供了更直接、更可控的制作流程。

新版默认保护原始表格和 Ren'Py 工程：导入后会在 workspace 中使用副本，关联工程只做扫描和预览，不会自动覆盖脚本或语音。所有写入工程的动作都需要用户明确点击应用。

### 本次重点

- **完整桌面工作流**：从导入 Excel 到生成 RPY、编辑表格、预览资源、合成语音、应用到工程，都可以在图形界面中连续完成。
- **安全的 workspace 模型**：软件内修改优先落在工作区副本中，避免直接污染原始表格；RPY 和语音同步到 Ren'Py 工程也改为手动应用。
- **新旧表格兼容**：继续支持旧版演出表，同时支持新版连续列结构；新版表格把立绘拆为左/中/右三列，转换时会按角色配置自动补回位置参数。
- **面向大表的编辑体验**：表格页支持多 sheet、搜索、冻结角色列、合理默认列宽、行增删插入、净差异追踪、大表缓存和切页保活。
- **资源导入与预览重做**：关联工程后，选中背景、立绘、音乐、音效单元格即可预览；导入或替换资源会写入 workspace，关联工程时同步写入工程资源目录。
- **语音合成工作台**：按 sheet 展示 TTS 行，支持单行、当前 sheet 和全局合成/应用；可试听、暂停、切换播放，并能撤销已生成但未应用的语音。
- **角色与语气统一配置**：角色启用、别名、立绘位置、语音指令和语气下拉集中管理；内置凉宫春日系列角色带默认模型、语气和立绘位置配置。
- **检查、对比和分发**：检查页按 sheet 与行号定位问题；对比页可独立导入两个表格并生成局部差异报告；提供 macOS 安装包和 Windows x64 免安装 zip。

### 重要细节

- 导入 Excel 后会自动保持转换结果最新，转换页不再需要手动点“转换”。
- 表格保存后，转换页会提示哪些 sheet 有未应用到工程的更改。
- 角色列留空会继承上一行角色，TTS 规划和语音文件命名也按这个规则处理。
- 语音状态会同时识别 workspace 与关联工程 `game/audio` 中已有的音频，避免已存在语音被误标为未生成。
- 新表立绘列可以只填资源名或“角色 编号”，位置参数由角色配置补齐；没有配置的角色默认使用 `left`、`mid`、`right`。
- 检查页不再把自定义立绘位置当作错误。

### 同步边界

- 关联 Ren'Py 工程只会扫描资源索引，不会自动覆盖工程文件。
- RPY 脚本只有点击“应用到工程”后才会写入 `game/*.rpy`。
- 语音合成先写入 pending 临时目录，点击“应用”后才会进入 workspace 的 `voice/`；如果已关联工程，会同时复制到 `game/audio/`。
- 图片、音乐、音效通过表格页导入或替换时会进入 workspace；已关联工程时也会同步进入工程资源目录。

### 如何使用？

- **Mac（Apple Silicon）** — 下载 `Excel2Rpy-1.0.0-arm64.dmg` 文件。
- **Windows 免安装版** — 下载 `Excel2Rpy-1.0.0-x64.zip` 文件，解压后运行 `Excel2Rpy.exe`。
- **Windows 安装版** — 下载 `Excel2Rpy-1.0.0-x64-setup.exe` 文件。
