## v1.1.0  更新日志

此版本主要优化工程维护、表格编辑和语音合成体验。

### 功能
- 新增工程页，集中提供关联工程后的高级工具。
- 新增音频归一化，支持 LUFS、峰值、RMS 三种标准。
- 新增资源体检，检查表格缺失资源和工程未引用资源。
- 表格编辑页和语音合成页新增全屏表格模式。
- 表格页和语音合成页支持点击行号互相定位。
- 语音合成页支持直接编辑语音文本并同步到表格。

### 优化
- 表格页和语音合成页会记住 sheet、语言和滚动位置。
- 语音合成页切回时不再整页刷新。
- 台词和语音文本支持在单元格内部展开查看。
- 方向键移动到台词或语音文本单元格时也会更新展开内容。
- 图片预览位置更稳定，方向键切换单元格时也会更新预览。
- 表格状态增加旧 workspace 兼容兜底，减少白屏风险。

### 构建
- 项目和发版流程切到 Node 24 LTS。
- Release Notes 读取仓库内归档文件，避免小标题被 Git 清理。

### 如何使用？

- **Mac（Apple Silicon）** — 下载 `Excel2Rpy-1.0.0-arm64.dmg` 文件。
- **Windows 免安装版** — 下载 `Excel2Rpy-1.0.0-x64.zip` 文件，解压后运行 `Excel2Rpy.exe`。
- **Windows 安装版** — 下载 `Excel2Rpy-1.0.0-x64-setup.exe` 文件。
